# タスク管理アプリ

日々のタスクを管理し、作業時間を計測できるアプリケーションです。

## 機能

- ✅ タスクの追加（色選択可能）
- ⏱️ ポモドーロタイマー（25分作業 + 5分休憩）
- 📊 タスクごとの時間計測と表示
- 📋 報告ボタンで作業内容をクリップボードにコピー
- 💾 タスク一覧の自動保存（ローカルストレージ）
- 📅 Googleカレンダー連携（今日のイベントを自動取得）

## 使い方

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

### ビルド

```bash
npm run build
```

## 使用方法

1. **タスクの追加**: タスク一覧の入力欄にタスク名を入力し、色を選択して「追加」ボタンをクリック
2. **時間計測の開始**: タスクをクリックするとポモドーロタイマーが自動で開始されます（25分）
3. **時間計測の停止**: 同じタスクを再度クリックすると停止します
4. **タスクの並び替え**: タスクをドラッグ&ドロップで移動できます
5. **報告の生成**: 「報告をコピー」ボタンをクリックすると、本日の作業内容がクリップボードにコピーされます

## Googleカレンダー連携

### セットアップ

1. [Google Cloud Console](https://console.cloud.google.com/)でプロジェクトを作成
2. Google Calendar APIを有効化
3. OAuth 2.0認証情報を作成（アプリケーションの種類: ウェブアプリケーション）
4. 承認済みのリダイレクト URIに `http://localhost:5173` を追加
5. クライアントIDをコピー

### 環境変数の設定

プロジェクトルートに `.env` ファイルを作成し、以下を追加：

```
VITE_GOOGLE_CLIENT_ID=your_client_id_here
```

### 使用方法

1. 「Googleカレンダーと連携」ボタンをクリック
2. Googleアカウントでログインして権限を許可
3. 「タスクを取得」ボタンで今日のイベントをタスクとして取得
